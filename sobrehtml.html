O que é HTML?

Linguagem de marcação usada para estruturar o conteúdo de páginas web no front-end, organizando elementos como textos, imagens, links e vídeos para que sejam visualizados nos navegadores como sites e aplicativos web. Ele funciona definindo uma hierarquia de conteúdo através de tags que criam seções, parágrafos, formulários e outros componentes interativos que o usuário vê na tela. No front-end, o HTML é essencial para construir a interface e a experiência visual do usuário.
