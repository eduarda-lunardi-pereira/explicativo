O que é HTML?

Linguagem de marcação usada para estruturar o conteúdo de páginas web no front-end, organizando elementos como textos, imagens, links e vídeos para que sejam visualizados nos navegadores como sites e aplicativos web.

Ele funciona definindo uma hierarquia de conteúdo através de tags que criam seções, parágrafos, formulários e outros componentes interativos que o usuário vê na tela.

O front-end (a parte do sistema que o usuário interage) é construído com HTML, CSS (para o estilo visual/roupagem) e JavaScript (para a interatividade e dinâmica). O navegador (Chrome, Firefox, etc.) lê o arquivo HTML e o renderiza, exibindo o conteúdo estruturado para o usuário.
